# Frontend - Gestor de Inversiones Personales

Este es mi TPFinal para la cursada de desarrollador Web Full stack para la UTN

Tecnologias que use

-   **React**
-   **Vite**
-   **TypeScript**
-   **Tailwind CSS** (use tailwind ya que se me hace mas practico y rapido a la hora de programar)
-   **Recharts** (use esta libreria para facilitarme hacer graficos para mi proyecto) 

Características

-   **Dashboard Interactivo** (cree un dashbord para cargar a la DB inversiones, modificarlas y/o eliminarlas llevando un conteo de las mismas)
-   **Gestión de Inversiones**: Hice todo un CRUD (Crear, Leer, Editar, Eliminar) de inversiones.
-   **Modo Oscuro/Claro**: (para los sensibles de vista como yo)
-   **Autenticación**: Cree autenticacion como fue pedida con validacion por JWT y nodemailer
